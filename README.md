# Project_Universal_AI

Для запуска нужно:
1) Создаём окружение: ```python3 -m venv venv```
2) Активируем окружение: ```source venv/bin/activate```
3) Обновим pip: ```pip install --upgrade pip```
4) Устанавливаем библиотеки: ```pip install -r requirements.txt```
5) Запускаем приложение: ```python3 main.py```

На тестовом сервере:
1) Активируем окружение: ```source venv/bin/activate```
2) Запускаем приложение: ```python3 main.py```

Датасет находятся по пути ```./data/raw/dataset.txt```
Результат попадает в ```./output/result.txt```
